TO HELP ABOUT A SPECIFIC COMMAND USE:
LOAD THEN PRINT "*<COMMAND>"

load/save commands
LOAD FAST - LOAD FROM SD
SAVE FAST - SAVE TO SD
LOAD SLOW - LOAD FROM TAPE
SAVE SLOW - SAVE TO TAPE
LOAD PEEK - 16 BIT PEEK
LOAD THEN POKE - 16 BIT POKE
LOAD THEN CLEAR - SET RAMTOP
LOAD THEN PRINT - TYPE A FILE

load* commands
128C - 128 USER DEF CHAR MODE
64C - 64 USER DEF CHAR MODE
BOLD - FORCE CHARS TO INV VIDEO
CD - CHANGE DIRECTORY
CP - COPY FILE
DEL - DELETE FILE
DIR - DIRECTORY LIST
FREE - SPACE IN SD
HEX - LOAD HEXA BLOCK INTO MEM
IN - IN IO PORT
INV - INVERT CHAR INTO A STRING
JOY - MAP KEYS TO JOYSTICK PORT
LDIR - COPY MEMORY BLOCK UP
LDDR - COPY MEMORY BLOCK DOWN
MAP - MAP MEM PAGE FOR DIR BLOCK
MC45 - ALLOWS EXEC AT 8000-BFFFH
MD - MAKE DIRECTORY
MV - MOVE/REN FILE
OPENDIR/ROW - DIR FROM A PROG
PEG - USER PROG SOUND EFFECT GEN
PLAY - PLAY 3 MUSIC CHANNELS
PWD - PRINT CUR DIR
OUT - OUT IO PORT
RTC - REAL TIME CLOCK
SAY - SPEECH 
USR - CALL TO MACHINE CODE
BAT - BATTERY
VER - FIRMWARE VERSION
VGM - PLAY A VGM FILE
VGMLOOP - SET VGM LOOP MODE
